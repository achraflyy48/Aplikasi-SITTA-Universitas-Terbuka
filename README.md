# Aplikasi SITTA

**Aplikasi SITTA** adalah website simulasi pemesanan bahan ajar Universitas Terbuka yang dibuat sebagai mini project untuk menyelesaikan tugas mata kuliah Pemrograman Berbasis Web.  
Proyek ini juga saya unggah ke GitHub sebagai bagian dari portofolio pribadi saya.

## ✨ Deskripsi Project

Aplikasi SITTA dirancang untuk mensimulasikan proses pemesanan bahan ajar oleh mahasiswa Universitas Terbuka.  
Melalui website ini, pengguna dapat melihat alur pemesanan bahan ajar secara sederhana, interaktif, dan mudah dipahami.

Project ini dibuat untuk melatih kemampuan dalam pengembangan aplikasi berbasis web, khususnya pada bagian:
- perancangan tampilan antarmuka,
- pengelolaan alur proses,
- serta implementasi fitur website secara fungsional.

## 🎯 Tujuan Project

- Menyelesaikan tugas mata kuliah Pemrograman Berbasis Web
- Membuat simulasi website pemesanan bahan ajar
- Melatih kemampuan membangun aplikasi berbasis web
- Menambah portofolio project di GitHub

## 🚀 Fitur Utama

- Tampilan halaman website yang sederhana dan user-friendly
- Simulasi proses pemesanan bahan ajar
- Navigasi antar halaman
- Form input data pengguna
- Ringkasan atau detail pesanan
- Desain yang responsif pada perangkat tertentu
